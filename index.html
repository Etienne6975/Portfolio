LE CODE S'ÉVEILLE

Dans l'univers infini du développement web, un jeune Padawan émerge de l'Université de Savoie, porteur d'un destin exceptionnel...

Armé des langages ancestraux HTML, CSS et JavaScript, ETIENNE MASSON se lance dans une quête épique pour maîtriser l'art du développement web et de la gestion des bases de données.

Guidé par la Force de la détermination et la sagesse de ses mentors, il forge son chemin à travers les défis numériques, maniant Laravel comme son sabre laser et PostgreSQL comme son bouclier.

De l'IUT d'Annecy jusqu'aux confins de l'univers digital, il construit des applications avec la précision d'un Jedi, respectant les principes SCRUM et les diagrammes UML comme un code d'honneur.

Son portfolio témoigne de ses exploits : du clone LEBONCOIN aux systèmes complexes, chaque projet est une victoire contre les bugs de l'Empire du Chaos.

L'aventure ne fait que commencer. La galaxie du code a besoin de développeurs courageux, adaptables et créatifs.

Que le Code soit avec vous, toujours.
